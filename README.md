# 🍕 Online Pizza Ordering System

A multi-page pizza ordering website developed using HTML, CSS, and JavaScript.

## Features

- Home, Menu, Services, Order, Contact Us, Sign In, and Logout pages
- Form validation using JavaScript
- Interactive user interface
- Consistent navigation across pages

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

- HTML for page structure
- CSS for styling
- JavaScript for validation and interactions

## Getting Started

1. Clone the repository:

git clone https://github.com/FauziyaPirji/pizza-ordering-system-js.git

2. Open the project folder.

3. Launch `index.html` in your browser.
